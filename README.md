# To-Do List

A simple to-do list web app built with React. Users can add tasks and remove tasks by clicking on them.

## Features
- Add new tasks
- Delete tasks (click to remove)
- Clean component structure (App, InputArea, ToDoItem)

## Tech Stack
- React
- JavaScript, ES6, HTML, CSS

## Running Locally

### Prerequisites
- Node.js + npm

### Steps to run locally

npm install

npm run dev
